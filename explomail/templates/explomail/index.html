A simple demo to facilitate discussion

6 nov 2018

Goals:

  • Make data and results accessible:
    • For non-programmers
    • For programmers
  • Be able to use many of the avaliable tools
  • Be able to implement new tools
  • Have a database structure and use its tools (searches, etc)
  • Be able to run on different machines

A first demo

An idea: a web application using django
  • Pros
    • Django is written in python
    • Easy to incorporate new and existing tools
    • Access to data:
      • Human-Data interaction through your browser
      • Machine-Data access through an API
      • For R you can use the httr library for example
    • Runs on Windows, Mac and Linux
    • Django implements several databases and search strategies
  • Cons
    • To use it you need to run a server locally, which makes starting up less friendly
